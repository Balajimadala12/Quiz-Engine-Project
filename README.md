# 🧠 Time-Based Quiz Application with Admin Leaderboard

## 📌 Project Description
This is a web-based Quiz Application developed using HTML, CSS, and JavaScript with Firebase as the backend database. 

The application includes a timer-based quiz system where users must complete questions within a limited time. Scores are calculated and stored in real-time.

An admin-specific feature is implemented to securely view the leaderboard of all participants.

---

🚀 **Live Demo**: https://baluquizweb.netlify.app


## 🚀 Features
- ⏱️ Time-based quiz system
- 🎯 Interactive question interface
- 📊 Real-time score calculation
- 🔐 Admin-only leaderboard access
- ☁️ Firebase database integration
- 💻 Clean and responsive UI

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend/Database: Firebase
